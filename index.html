Portafolio (Proyectos destacados)

Desarrollo de una alerta de pánico para notificar eventos de robo

Android + Java Backend

Fecha: 31/05/2019

Puedes verlo en: DOI

Prototipo de alarma de pánico a través de una aplicación móvil para teléfonos inteligentes Android, capaz de notificar incidentes de robos al Servicio Integrado de Seguridad ECU 911 y a una aplicación desarrollada en Java dentro de un servidor. Dicha alarma de pánico la activa el usuario a través de una combinación de botones físicos del teléfono inteligente, en ese instante se envía la información del usuario al ECU911 y al servidor implementado, el cual activará un clúster de cámaras para videovigilancia y enviará una notificación por correo electrónico a una persona encargada del monitoreo, quien podrá observar a través de un enlace lo que está sucediendo en el lugar de los hechos y poder de alguna manera socorrer a la víctima.
